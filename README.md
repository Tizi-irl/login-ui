# login-ui
A modern and responsive login interface built with HTML and CSS.

## About
This project is a beginner-friendly frontend login page. It focuses on layout design, styling, and basic UI structure.

## Features
- Clean and modern design
- Responsive centered layout
- Email and password input fields
- Social login buttons (Google and Apple style)

## Built With
- HTML5
- CSS3

## Purpose
This project was created for learning and portfolio purposes.
